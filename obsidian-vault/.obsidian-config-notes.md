# Obsidian Setup — Carlos's Dell Windows 11 Machine

## Step 1: Install Obsidian
Download from: https://obsidian.md
Install on your Dell 5440.

## Step 2: Clone the Hermes repo
Open PowerShell or Git Bash:
  git clone https://github.com/Daaanovaaa/hermes-ccnow.git C:\Users\[YourName]\hermes-ccnow

## Step 3: Open the vault in Obsidian
1. Launch Obsidian
2. Click "Open folder as vault"
3. Navigate to: C:\Users\[YourName]\hermes-ccnow\obsidian-vault
4. Click Open

## Step 4: Install Obsidian Git plugin
1. In Obsidian: Settings → Community plugins → Browse
2. Search "Obsidian Git"
3. Install and enable
4. Settings → Obsidian Git:
   - Auto pull interval: 30 (minutes)
   - Auto push on commit: enabled
   - Pull on startup: enabled

## Step 5: Test
The vault will auto-pull from GitHub every 30 minutes.
Each morning, HOME.md will show your daily dashboard with:
- Active case count
- Budget status
- Upcoming deadlines
- Latest case log entries
